# h5-ca28-archive

## 项目简介

h5-ca28-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 页面的集中存储与展示空间，方便历史版本查阅、页面存档或简单的前端页面托管。

## 目录说明

```
h5-ca28-archive/
├── pages/          # 存放各独立 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── index.html      # 仓库页面索引（可选）
└── README.md       # 本文件
```

- **pages/**：包含所有归档的 HTML 页面，按功能或时间分类存放。
- **assets/**：存放页面引用的公共资源文件，保持目录结构清晰。
- **index.html**：可选的首页索引，用于快速导航至各页面。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无强依赖关系，资源引用均使用相对路径。
- 归档页面可能包含示例数据、原型设计、学习笔记等，不保证内容持续更新。
- 部分页面可能依赖外部 CDN 资源，离线查看时可能出现加载失败的情况。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/h5-ca28-archive.git
   ```
2. 打开 `pages/` 目录下的任意 HTML 文件即可查看内容。
3. 若需在线访问，可启用 GitHub Pages 并设置根目录为仓库根路径。
